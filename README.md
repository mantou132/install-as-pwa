# Install as PWA

Convert any website into installable PWA.

## How to work

- intercept document and inject WebManifest, ServiceWorker
- intercept WebManifest, ServiceWorker, and response

## development

```bash
# install dependencies
npm install
# start develop
npm run start
```

Build:
```bash
npm run build:zip
```
